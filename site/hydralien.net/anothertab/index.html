AnotherTab A Google Chrome, Mozilla Firefox and Microsoft Edge extension to simplify bookmarks and extensions access via New Tab page

About

This extension replaces browser standard New Tab page with a very simple thing - a list of a content of a selected bookmarks folder (Bookmarks Bar by default), installed browser extensions list and some useful browser shortcuts like settings, passwords, cookies, extensions and data cleanup.

It also has an option to highlight, delete or rearrange bookmarks within selected folder to make its location easy to remember.

Additionally, several customization options are available, such as background color/image selection, tile size choice, tile-specific highlighting and selection of which directory to use as bookmarks root.

Extension also uses synchronised browser storage API to store settings locally and remotely.
